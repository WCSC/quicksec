quicksec
========

IDAPython script for quick vulnerability analysis

Just run the script. Vulnerable libc function calls will be marked out in the idb for you. 

Currently supported functions: 
 * `gets`
 * `fgets`
 * `strcpy`
 * `recv`
 * `read`
 * `printf`
